===============================
  FROGMAGEDDON
  Get rid of the frogs invading the office!
===============================

HOW TO PLAY:
1. Double-click Frogmageddon.exe
2. Your browser will open automatically with the game
3. A console window will stay open in the background — that's normal
4. When you close the browser tab, the console window will close itself after a few seconds

CONTROLS:
- WASD: Move
- Mouse + Click: Shoot
- ESC: Pause

NOTES:
- Windows may ask to allow network access the first time — it's only using localhost
  (your own computer), not the internet.
- If your browser doesn't open automatically, look in the console window for the URL
  and paste it into your browser manually.
- First launch may be slightly slower as Windows caches the app files.

Have fun!
